Bot API
=======

В этом разделе представлена справочная информация по фреймворку **maxo**.
Здесь вы найдёте описание всех ключевых компонентов, необходимых для взаимодействия с платформой max.ru.

Содержание раздела:

* :doc:`bot` – основной класс для управления ботом, выполнения запросов и настройки параметров сессии.
* :doc:`types` – полный список моделей данных и объектов, используемых в API (сообщения, пользователи, чаты и др.).
* :doc:`methods` – справочник всех доступных методов для взаимодействия с платформой (отправка текста, медиафайлов, управление клавиатурами).
* :doc:`enums` – перечисления и константы, обеспечивающие строгую типизацию параметров и событий.
* :doc:`updates` – модели обновлений (событий), получаемых ботом (``MessageCreated``, ``MessageCallback`` и др.).
